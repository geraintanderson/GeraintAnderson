Palysade

While working at GB3 I designed and built a safeguarding application. I built the application as a full stack JavaScript application using AngularJS, Express and MySQL.

http://palysade.co.uk/

This is used at a large dioceses in England and we received excellent feedback from the customers. My intention was to create an application that's easy to use becuase many of the people who work in safeguarding are not very tech savvy. Using a relational database fitted the data model well where people can navigate the application in many different ways leading to an intuitive user experience.

I built this in a way that would allow other customers to sign up with minimal re-work allowing it to be re-branded and used for other customers.
