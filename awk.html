Awk

Syntax

#!/usr/bin/nawk –f First line of a directly executable NAWK program. Without this line the command calling the script will need tell the shell that awk will run the script as below. With this line the script can be called with only the script name and location.

Options

-v name=value pass in a variable

-F"c" Set the delimiter to a character

Commands

“UNIX command”|getline variable Run a unix command in awk

variable++ --variable variable1*=variable2 /= -= ^= %= Modify a variable value

/^$Blank line regular expression

Functions

systime() Returns UNIX time, seconds since epoch(01-01-1970 00:00:00).

sub(/regexp/,str,var) substitute the first occurrence of regexp with str in the variable var.

gsub(/regexp/,str,var)substitute all occurrences of regexp with str in the variable var.

length(var) Returns the length of the variable

mktime(YYYY MM DD H M S) Change the date to Unix time for date calculations mktime(“2014 08 21 0 0 0”)

strftime("%c", time)Convert the date format from unix time to local format strftime("%c", 1234567890)

Variables

FSField Separator

NFNumber of Fields

NRRecord number
