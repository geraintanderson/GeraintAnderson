f = open(filename, 'mode')Open a file. The mode is optional, blank being read only. Available modes are r - read r+ - reading and writing w - write a - append b - Windows only, append to the mode to open in binary

f.tell()Returns the file objects current position in the file

f.seek(offset, from_what) Change the file objects position by offsetting from a reference point. From what is the reference point and uses the values: 0 - Beginning of the file 1 - Current file position 2 - End of the file

f.close() Close the file
