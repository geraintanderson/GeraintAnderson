JavaScript

Best Practices

IIFE's are used to avoid creating global functions. These are self calling function. (function(){/*Some function code*/}())

Syntax

var variable Define a variable to use in the current scope. Without the keyword “var” the variable is defined globally. It can be assigned a value at the same time using var variable = value

variable = { property: value } Assign a javascript object with property a to the variable

variable.property Access the property of a variable

variable[property] Access the property of a variable

“string” Double quotes or single quotes work however double is usually preferred because it aligns with the JSON standard

Angular

Syntax

Angular.js pages must reference the Angular.js file and contain one ng-app directive which is usually included in the html tag.

JQuery

Syntax>

$(selector).action() Basic JQuery syntax

$(selector).action1().action2()Chain commands for increased performance so the browser doesn’t need to get the same element multiple times.

$(document).ready(function(){ // jQuery methods go here... }); Wait until the document is fully loaded before doing stuff to avoid letters by performing actions on selectors that don’t yet exist

$(function( ng-show ng-hide){ // jQuery methods go here... }); Same as the document ready function above to hold off execution until the document is loaded. This alternative may be less readable.

$(“*”)Selects all elements

$(this)Selects the current HTML element

$(“element.class”) Selects all element with a certain class e.g. p.intro is all paragraph elements with class intro.

$(“element:first”) Select the first element

$(“[attribute]”)Select all elements with attribute e.g. $(“[href]”)

$("a[attribute=value]")Select all elements with an attribute with the value value

$(“element1, element2)Select multiple elements
